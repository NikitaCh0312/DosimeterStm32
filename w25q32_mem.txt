для стирания возможно:
  всего - 1024 сектора  
  1 сектор - 4096 байт (за раз)

для записи 
  за 1 раз можно записать до 256 байт (1 станица)

Протокол памяти:
  1- сектор для хранения конфигов работы прибора
    протокол:
      - версия прошивки + мак + ip где то сверху (структура )

  2-4 сектор для хранения карт заданий
    протокол:
      - одна карта 252 + црц16 = 254 байта
      - на каждую страницу по одной структуре карты
      итого до 48 карт держим в памяти

  5-7 сектор для зранения указателей на логи
    протокол:
      - один указатель: 1 байт = год; 1 байт месяц; 4 байта адрес; 2 байта црц16, итого 8 байт
      - всего 32 указателя на страницу, 512 на сектор, до 1536 указателей всего (128 лет без сброса батарейки)
  остальное логи
